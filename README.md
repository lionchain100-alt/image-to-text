# Image to Text Converter

Free online OCR tool built with Next.js + Tesseract.js. Deployed on Cloudflare Pages.

## Features

- 🔒 Privacy-first: all processing in the browser, no uploads
- ⚡ Free & unlimited
- 🌍 100+ languages via Tesseract.js
- 📋 Drag & drop, click, or paste (Ctrl+V)

## Tech Stack

- Next.js 14 (static export)
- Tesseract.js v5
- Tailwind CSS
- Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# Upload /out folder to Cloudflare Pages
```
